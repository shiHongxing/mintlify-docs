# 更新日志


## 2024年7月12日 星期五

- 新功能

```
1、新增设置自动停止，避免因忘记关机而产生大额费用
2、新增卡券规则说明以及适用产品说明
```

- 优化项

```
1、裸金属云市场新增“立即咨询”和“联系客服”按钮，便于实际消费用户能够联系我们
2、优化克隆实例功能，次数限定为：每日克隆成功实例为3次
3、优化账单详情页面，增加单价信息，用户可以更清楚的了解自己的消费信息
4、优化云存储页面，传输列表弹框升级为抽屉弹框展示，提升用户使用体验
5、优化miniconda、pytorch、TensorFlow三种框架的镜像，以解决在执行 Python 时必须指定版本号的问题
```



## 2024年7月5日 星期五

- 新功能

```
新增应用模式，提供的当下热门AI应用，无需自己搭建环境，支持一键启动快速部署。
新增【应用市场】&【应用详情】，罗列了用户感兴趣的应用（如果没有您的所需应用，请联系客服，我们会尽快满足您的需求。
新增【我的应用】，包含工作空间、应用启动器、镜像仓库以及云存储。
新增【工作空间】，展示用户部署的应用，在使用应用之外，用户也可以使用创建该应用的实例，做到算力资源的充分使用。
新增【应用启动器】，同应用市场，提供多个应用入口供用户方便使用。
新增【镜像仓库】，为您提供一个集中的位置来存储和管理镜像。
新增【云存储】，您可以轻松存储、访问和共享海量数据，而且还享有高级安全保障和灵活的存储调整选项。让您的数据管理更加高效和便捷！
```




## 2024年6月7日 星期五

- 新功能

```
新增站内信强提醒消息，避免用户错过重要信息
新增短信、邮件、微信服务号提醒信息，包含登录、充值提醒；
新增产品动态，可在【文档】-【产品动态】关注智算云平台成长的每一步。
新增账号安全检测，非法号码需完成实名认证后才可消费。
新增【保存镜像】进度条显示，提升用户产品体验感。
```

- 优化项

```
增加平台安全性检测，针对境外账号需通过图形验证后方可购买实例
优化包年包月实例到期之后无法开机使用的难题，包年包月到期之后可以按照按量计费的形式开机
```
